# Jazira Flow

This project renders a small flow editor prototype using D3 and Vite.

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server with hot module replacement:

   ```bash
   npm run dev
   ```

3. Create an optimized production build:

   ```bash
   npm run build
   ```

4. (Optional) Preview the production build locally:

   ```bash
   npm run preview
   ```

The UI loads at the dev server URL (default `http://localhost:5173/`). Use the panel buttons to add nodes, randomize layout, stabilize the force simulation, export, and import flow definitions.
